# CV Maker - Professional Resume Builder

CV Maker is a modern, user-friendly web application designed to help users create professional resumes in minutes. With real-time preview and multiple expert-designed templates, building a standout CV has never been easier.

![CV Maker Demo](public/favicon.ico)

## Features

- **Multiple Professional Templates**: Choose from 6 different styles:
  - **Classic**: Timeless and professional for corporate roles.
  - **Modern**: Bold and creative for tech and design.
  - **Minimal**: Clean and simple for content-focused resumes.
  - **Executive**: Elegant and sophisticated for senior positions.
  - **Creative**: Vibrant and colorful for creative industries.
  - **Technical**: Code-inspired design for developers.
- **Real-time Preview**: See your changes instantly as you type.
- **Multilingual Support**: Available in both **English** and **Portuguese**.
- **PDF Export**: Download your finished CV as a high-quality PDF.
- **Privacy First**: No user data is stored on our servers; everything stays in your browser.
- **SEO Optimized**: Built with SEO best practices for better discoverability.
- **Cookie Transparency**: Explicitly informs users about data storage (or lack thereof).

## Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS 4.x](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Navigation**: [React Router 7](https://reactrouter.com/)
- **Head Management**: [React Helmet Async](https://github.com/staylor/react-helmet-async) (SEO)
- **PDF Generation**: Native Browser Print (optimized with Custom CSS)

## Getting Started

### Prerequisites

- Node.js (version 18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/PatrickPontes44/cvmaker.git
   ```

2. Navigate to the project directory:
   ```bash
   cd cvmaker
   ```

3. Install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```
   *Note: Using `--legacy-peer-deps` is recommended to avoid version conflicts with the latest React/Router versions.*

### Running the Project

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.
